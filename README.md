Welcome!

Prerequisites:

· Android Studio Iguana or newer
· Android SDK API 24+
· Kotlin 1.9.0+

Installation:

1. Clone the repository
   git clone https://github.com/Zanghai07/Zanghai-Music.git
2. Open in Android Studio
   · Select "Open an existing project"
   · Navigate to the cloned directory
3. Build the project
   · Let Gradle sync and download dependencies
   · Click Build > Make Project
4. Run the app
   · Connect your device or start an emulator
   · Click Run > Run 'app'
   
Built With:

· Kotlin - Primary programming language
· Coroutines - Asynchronous programming
· Glide - Image loading and caching
· LyricViewX - Lyrics display (Moriafly)
· Material Components - UI components

How to Use:

1. Add Music: Place your audio files in /storage/emulated/0/zanghai/ folder
2. Open App: Launch Zanghai Music
3. Scan: App automatically scans the folder for music files
4. Play: Select a song and enjoy!
5. Lyrics: App will automatically fetch lyrics for playing songs
